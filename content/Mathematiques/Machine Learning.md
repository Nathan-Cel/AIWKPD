# Le Machine Learning (Apprentissage Automatique)

Le **Machine Learning** est un sous-domaine de l'[[Intelligence Artificielle]] qui donne aux ordinateurs la capacité d'apprendre à partir de données, sans être explicitement programmés pour une tâche spécifique.

L'objectif est de créer des [[Modèle de Machine Learning|modèles]] capables de faire des prédictions ou de prendre des décisions après avoir été "entraînés" sur un [[Jeu de données]].

### Les grandes familles d'apprentissage :
- **[[Apprentissage supervisé]] :** Apprendre à partir de données étiquetées (ex: [[Régression]], [[Classification]]).
- **Apprentissage non supervisé :** Trouver des structures dans des données non étiquetées (ex: clustering).
- **Apprentissage par renforcement :** Un agent apprend à agir dans un environnement pour maximiser une récompense.
